# 门店地图功能说明

## 功能概述

门店地图功能用于在地图上展示所有门店的位置，用户可以：
- 查看所有门店列表
- 在地图上查看门店位置
- 点击门店标记查看详细信息
- 点击门店卡片定位到地图上的位置

## 技术实现

### 后端
- **数据库表**：`tb_store`（门店表）
- **实体类**：`com.fuding.entity.Store`
- **接口**：
  - `GET /api/store/list` - 获取门店列表（支持按城市筛选）
  - `GET /api/store/{id}` - 获取门店详情
  - `GET /api/store/nearby` - 获取附近门店（根据经纬度）

### 前端
- **页面路径**：`/stores`
- **组件位置**：`src/views/Stores.vue`
- **地图API**：高德地图 JS API 2.0

## 配置高德地图API Key

### 1. 申请高德地图API Key

1. 访问 [高德开放平台](https://lbs.amap.com/)
2. 注册/登录账号
3. 进入控制台 → 应用管理 → 我的应用 → 创建新应用
4. 添加 Key：
   - 服务平台：选择"Web服务"或"Web端(JS API)"
   - 应用名称：填写你的应用名称（如：福鼎白茶服务平台）
5. 获取生成的 Key

### 2. 配置API Key

#### 方式一：使用环境变量（推荐）

在 `frontend` 目录下创建或编辑 `.env` 文件：

```env
# 高德地图API Key
VITE_AMAP_KEY=你的高德地图API_Key
```

#### 方式二：直接修改代码

如果不想使用环境变量，可以直接修改 `src/views/Stores.vue` 文件：

找到这一行：
```javascript
const amapKey = import.meta.env.VITE_AMAP_KEY || 'YOUR_AMAP_KEY'
```

替换为：
```javascript
const amapKey = '你的高德地图API_Key'
```

### 3. 重启开发服务器

配置完成后，需要重启前端开发服务器：

```bash
cd frontend
npm run dev
```

## 数据说明

### 门店数据

当前系统中的门店数据为**虚构示例数据**，仅用于课程设计演示，不代表真实商户。

门店数据特点：
- 基于福鼎市真实地理坐标
- 门店名称、联系方式等信息均为虚构
- 共15个示例门店，分布在福鼎市不同区域

### 数据来源声明

在门店地图页面顶部已添加提示信息：
> "本系统为课程设计演示，门店数据为虚构示例，不代表真实商户"

## 功能特性

### 1. 门店列表
- 左侧显示所有门店卡片
- 显示门店名称、地址、电话、营业时间
- 点击卡片可定位到地图上的位置

### 2. 地图展示
- 使用高德地图展示门店位置
- 每个门店在地图上显示为标记点
- 点击标记点显示门店详细信息窗口

### 3. 交互功能
- **定位功能**：点击门店卡片的"定位"按钮，地图会自动移动到该门店位置
- **信息窗口**：点击地图上的标记点，显示门店详细信息
- **高亮显示**：选中的门店卡片会高亮显示

## 注意事项

1. **API Key限制**：
   - 高德地图个人开发者有免费调用额度
   - 超出额度后需要付费或升级账号
   - 建议在开发时控制调用频率

2. **网络要求**：
   - 地图API需要网络连接
   - 如果API Key未配置或配置错误，地图功能将无法使用
   - 门店列表功能不受影响，仍可正常查看

3. **坐标系统**：
   - 使用高德地图坐标系（GCJ-02）
   - 数据库中的经纬度数据基于高德地图坐标

4. **浏览器兼容性**：
   - 支持现代浏览器（Chrome、Firefox、Edge、Safari等）
   - 需要JavaScript支持

## 扩展功能建议

## 常见问题

### Q: 地图无法显示？
A: 检查以下几点：
1. 是否配置了高德地图API Key
2. API Key是否正确
3. 网络连接是否正常
4. 浏览器控制台是否有错误信息

### Q: 门店数据如何修改？
A: 可以通过以下方式：
1. 直接在数据库中修改 `tb_store` 表
2. 通过管理后台添加门店管理功能（需要开发）

### Q: 如何添加更多门店？
A: 在数据库 `tb_store` 表中插入新记录，注意：
- 经纬度需要使用高德地图坐标
- 可以使用[高德地图坐标拾取工具](https://lbs.amap.com/tools/picker)获取坐标
