# 前端素材替换位置说明

## 📁 素材存放目录

### 1. 静态资源目录：`frontend/public/`
所有静态资源（图片、图标等）应放在 `public` 目录下，访问路径为 `/文件名`

**需要创建的文件：**
```
frontend/public/
├── favicon.ico              # 网站图标（浏览器标签页图标）
├── banner1.jpg              # 首页轮播图1
├── banner2.jpg              # 首页轮播图2（可添加更多）
├── default-product.png      # 默认产品图片（产品无图时显示）
└── default-category.png     # 默认分类图片（分类无图时显示）
```

## 🖼️ 需要替换的素材位置

### 一、首页素材（`src/views/Home.vue`）

#### 1. 轮播图
**位置：** `src/views/Home.vue` 第 67-69 行
```javascript
const banners = ref([
  { id: 1, title: '福鼎白茶', image: '/banner1.jpg' },
  { id: 2, title: '优质茶叶', image: '/banner2.jpg' }
])
```

**替换方法：**
- 将轮播图片命名为 `banner1.jpg`、`banner2.jpg` 等，放入 `public` 目录
- 如需添加更多轮播图，在数组中添加新项
- 建议尺寸：1920x600px 或 1920x400px

#### 2. 分类图片
**位置：** `src/views/Home.vue` 第 24 行
```vue
<img :src="category.image || '/default-category.png'" :alt="category.name" />
```

**替换方法：**
- 分类图片由后端API返回，如果分类没有图片，会显示默认图片
- 默认图片：将 `default-category.png` 放入 `public` 目录
- 建议尺寸：300x200px

#### 3. 产品图片
**位置：** `src/views/Home.vue` 第 37 行
```vue
<img :src="product.image || '/default-product.png'" :alt="product.name" />
```

**替换方法：**
- 产品图片由后端API返回，如果产品没有图片，会显示默认图片
- 默认图片：将 `default-product.png` 放入 `public` 目录
- 建议尺寸：400x400px

### 二、产品列表页（`src/views/Products.vue`）

**位置：** `src/views/Products.vue` 第 30 行
```vue
<img :src="product.image || '/default-product.png'" :alt="product.name" />
```

**替换方法：**
- 使用与首页相同的默认产品图片

### 三、产品详情页（`src/views/ProductDetail.vue`）

<el-image :src="product.image || '/default-product.png'" fit="contain" style="width: 100%;" />

**替换方法：**
- 使用与首页相同的默认产品图片

### 四、购物车页面（`src/views/Cart.vue`）

```vue
<img :src="scope.row.product?.image || '/default-product.png'" :alt="scope.row.product?.name" />
```

**替换方法：**
- 使用与首页相同的默认产品图片

### 五、订单页面（`src/views/Orders.vue`）

**位置：** `src/views/Orders.vue` 第 24 行
```vue
<img :src="item.productImage || '/default-product.png'" :alt="item.productName" />
```

**替换方法：**
- 使用与首页相同的默认产品图片

### 六、用户头像（`src/components/Header.vue`）

**位置：** `src/components/Header.vue` 第 24 行
```vue
<el-avatar :src="userInfo?.avatar" :size="30">{{ userInfo?.nickname?.charAt(0) }}</el-avatar>
```

**替换方法：**
- 用户头像由后端API返回
- 如果用户没有头像，会显示用户昵称的首字母

### 七、网站图标（`public/index.html`）

**位置：** `public/index.html` 第 6 行
```html
<link rel="icon" href="<%= BASE_URL %>favicon.ico">
```

**替换方法：**
- 将网站图标命名为 `favicon.ico`，放入 `public` 目录
- 建议尺寸：32x32px 或 64x64px

## 🎨 样式相关素材

### 登录/注册页面背景渐变
**位置：** 
- `src/views/Login.vue` 第 99 行
- `src/views/Register.vue` 第 123 行

```css
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
```

**替换方法：**
- 如需使用背景图片，可修改为：
```css
background-image: url('/login-bg.jpg');
background-size: cover;
background-position: center;
```
- 然后将背景图片 `login-bg.jpg` 放入 `public` 目录

## 🔧 素材替换步骤

1. **准备素材文件**
   - 按照建议尺寸准备图片
   - 确保图片格式正确（jpg、png等）

2. **放入public目录**
   - 将所有静态素材文件放入 `frontend/public/` 目录

3. **修改代码（如需要）**
   - 如需添加更多轮播图，修改 `Home.vue` 中的 `banners` 数组
   - 如需修改背景，修改对应组件的样式

4. **测试验证**
   - 启动前端项目，检查所有图片是否正常显示
   - 检查默认图片是否在无图时正确显示

## 📐 推荐图片尺寸

| 素材类型 | 推荐尺寸 | 格式 |
|---------|---------|------|
| 轮播图 | 1920x600px | JPG |
| 产品图片 | 400x400px | PNG/JPG |
| 分类图片 | 300x200px | PNG/JPG |
| 网站图标 | 32x32px 或 64x64px | ICO |
| 用户头像 | 200x200px | PNG/JPG |
| 登录背景 | 1920x1080px | JPG |

## ⚠️ 注意事项

1. **文件命名**：确保文件名与代码中引用的路径一致（区分大小写）
2. **文件大小**：建议压缩图片，避免影响加载速度
3. **图片格式**：
   - 照片类使用 JPG
   - 图标、Logo使用 PNG（支持透明）
   - 网站图标使用 ICO
4. **路径引用**：`public` 目录下的文件使用 `/文件名` 访问，不需要 `public` 前缀
5. **动态图片**：产品、分类等图片由后端API返回，需要在后端管理系统中上传
